# Отчётные документы по итогам тестирования
## 1. Краткое описание

В ходе выполнения дипломной работы было проведено автоматизированное тестирование приложения aqa-shop.jar,
с поддержкой эмулятора банковских сервисов, расположенного в gate-simulator, взаимодействующий с СУБД (mySQL и PostgreSQL).

1. Был составлен [план автоматизации](https://github.com/SofyaDavydova/Diploma/blob/main/documents/Plan.md);
2. Согласно плану было проведено автоматизированное тестирование функционала покупки и оформления кредита по данным карты веб-сервиса покупки "Путешествие дня".
3. Были составлены отчеты о проведенном тестировании.

## 2. Количество тест-кейсов

Общее колличество тест-кейсов - 94:
+ Оплата по кнопке "Купить" - 43
+ Оплата по кнопке "Купить в кредит" - 43
